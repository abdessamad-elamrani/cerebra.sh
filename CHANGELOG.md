# Changelog

## v2.0.0

Initial public release.

- Multi-protocol workspace: SSH, RDP, VNC, and local shell in one interface
- Runbooks with direct pane injection and marketplace
- AI Toolkit with live workspace context and AI-Cyber Mode
- CyberViewer for attack-flow graph visualization
- Cherry Notes for evidence capture
- Intel module for indicator enrichment (domains, IPs, URLs, hashes)
- RevEng module for binary triage, API-to-ATT&CK mapping, deobfuscation, IoC extraction
- Crypto module for JWT decoding, hashing, and format transforms
- Available on macOS (ARM64), Windows (x64), and Linux (x64 AppImage)
